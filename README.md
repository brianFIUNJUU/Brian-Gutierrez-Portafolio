# Brian-Gutierrez-Portafolio
Sistema de Novedades Policiales: Plataforma centralizada para el
registro, seguimiento y gestión de eventos e incidencias policiales.

 Sistema de Auditoría de Documentación Policial: Herramienta de
control diseñada para garantizar la trazabilidad, autenticidad y
gestión documental para el armado de legajo policial.

 Legajo Prontuarial Ciudadano: Sistema de gestión digital de
historiales, optimizado para la consulta ágil y actualización de
datos personales, huellas dactilares,etc.

 Sistema de Vigilancia Integrado: Desarrollo de una plataforma de
monitoreo que utiliza OpenStreetMap para el mapeo geoespacial,
permitiendo la ubicación precisa de puntos de vigilancia y
visualización estratégica de recursos.
